# 键的排序
在一个字典的元素中我们确实需要强调某种顺序:

## 1.sort()
- 通过字典的keys方法收集一个键的列表
- 使用列表的sort方法进行排序
- for循环逐个进行显示结果

```python
Ks list(D.keys()) # 通过字典的keys方法收集一个键的列表  >>>Ks ['a','c','b']

Ks.sort()  # 使用列表的sort方法进行排序  >>>Ks ['a','b','c']

for key in Ks: print(key,'=>',D[key]) # for循环逐个进行显示结果
```
