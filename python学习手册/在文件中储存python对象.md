# JSON格式储存

--> 标准库

JSON是一种新兴的数据交换格式,它既是语言无关的,也支持多种系统。

JSON与Python中的字典和列表在语法上的相似性， 因此Python的json标准库模块能够很容易地在Python对象与JSON格式之间来回转换
