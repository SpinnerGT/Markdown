# 注意
- 赋值的变量名除非被声明为global或nonlocal,否则均为局部变量。

- 函数的每次调用都会创建一个新的局部作用域

```ad-tip
title: 原位置改变对象

原位置改变对象并不会把变量划分为局部变量，实际上只有对变量名赋值才可以。

	例如，如果变量名L在模块的顶层被赋值为一个列表，在一个函数内类似L.append(X)  这样的语句并不会将L划分为局部变量，而L=X却可以


```

# global(尽可能避免)
在函数类直接改变全局变量:
```jupyter
X = 88

def func():
	global X # 呼叫外部X
	X = 10

# 外部X变为10
```
