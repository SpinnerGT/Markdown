# 语法
`<input type="radio" name="组名" value="取值" />`

- name属性表示单选按钮所在的组名
![[name属性]]
- value表示单选按钮的取值



![[Pasted image 20220214230014.png]]


# 默认选着:checked

```
<input type="radio" name="gender" value="男" checked />男

<input type="radio" name="gender" value="男" checked="checked" />
```
