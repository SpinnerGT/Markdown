# button

普通按钮一般情况下都是配合JavaScript来进行各种操作的
`<input type="button" value="取值" />`

> value的取值就是按钮上的文字。



# submit
提交按钮一般都是用来给服务器提交数据的

`<input type="submit" value="取值" />`
> value的取值就是按钮上的文字。


# reset

重置按钮一般用来清除用户在表单中输入的内容

``<input type="reset" value="取值" /``
> value的取值就是按钮上的文字。

---

> 重置按钮只能清空它“所在form标签”内表单中的内容，对于当前所在form标签之外的表单清除是无效的。
