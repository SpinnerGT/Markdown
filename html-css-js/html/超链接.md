# 超链接hyperlink


## a标签

```
<a href="链接地址">文本或图片</a>
```

- 文字超链接

- 图片超链接
```
<a href="http://www.lvyestudy.com"><img src="img/lvye.png" alt="绿叶学习网"/></a>
```


## target属性



target属性来定义超链接打开窗口的方式
```
<a href="链接地址" target="打开方式"></a
```

## 内部链接
位于同一个网站根目录下的

> 内部链接使用的都是相对路径


### 锚点链接

指向的是当前页面的某个部分

- id 唯一
- href   `#id`

```
<a href="#id">推荐文章</a>
....

<div id='123'>
...
</div>
```
